# Images Directory

此目录用于存放小程序所需的图片资源。

## 需要准备的图片

### TabBar 图标
- `tab-home.png` - 首页图标 (81x81px)
- `tab-home-active.png` - 首页选中图标 (81x81px)
- `tab-exercise.png` - 动作库图标 (81x81px)
- `tab-exercise-active.png` - 动作库选中图标 (81x81px)
- `tab-profile.png` - 我的图标 (81x81px)
- `tab-profile-active.png` - 我的选中图标 (81x81px)

### 占位图
- `empty.png` - 空状态占位图
- `default-avatar.png` - 默认头像
- `logo.png` - 应用Logo

> 提示: 图标建议使用 PNG 格式，大小 81x81px，线条简洁清晰。
